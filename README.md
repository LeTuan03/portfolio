# Cinematic Portfolio

A high-end, cinematic portfolio website built with Next.js, TailwindCSS, Framer Motion, and React Three Fiber.

## Tech Stack

- **Framework**: Next.js 15+
- **Styling**: TailwindCSS
- **Animation**: Framer Motion
- **3D**: React Three Fiber + Drei
- **Icons**: Lucide React

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run the development server**:
    ```bash
    npm run dev
    ```

3.  Open [http://localhost:3000](http://localhost:3000) with your browser.

## Features

- **Hero Section**: 3D particle system and floating shapes.
- **Smooth Animations**: Scroll reveals, magnetic buttons, and parallax effects.
- **Glassmorphism**: Premium UI cards with blur effects.
- **Responsive**: Fully optimized for mobile and desktop.
